Collecting and Labeling Data

N ow that we had our idea chosen out, it was time to collect data and label it. We first started off by downloading many images of 9 different crystal types: Quartz, Rose Quartz, Amethyst, Garnet, Celestine, Turquoise, Pyrite, Corundum and Malachite. Then, we looked through the different images we downloaded and deleted all the ones that didnâ€™t apply. We then inputted all the data into LabelBox in order to label them accordingly based on the type of crystal. Our main goal was to find the best data possible, which meant we tried our best by labeling images that are as high-resolution as possible in order for our model to be accurate. Finally, after 2 days of labeling data, we moved it all into a json file where we were ready for the data processing.

Data Processing

T eam 1 - Downloading and Labeling Data
Shivani and Jaedyn
O ur job was to download and upload the l abeled data from a json file from LabelBox into a file in order for Team 2 to analyze it. We used Pandas and Python code in order to extract all labeled (not skipped) images into a .txt file where we named it using a specific naming convention for Team 2 to use. It took us a bit and some trial and errors, but we finally moved all 729 pictures into a .txt file successfully!

T eam 2 - Data Formatting and Anaylsis
Arnav and Varini
O ur job was to calculate and organize the dimensions for each label and picture so that they pair up accordingly. For every image, we had to create a separate .txt file and take the downloaded pictures and sort them through by label dimensions. We then had to put the properties of each label in it so that Team 3 has an easier time understanding it. It worked successfully! 

Team 3 - Distributing the Data
Chris and Winston
Our job wa s to take all of the output from Teams 1 and 2. After that, we had to code to divide the data in which 80% would be used for training, 10% for testing and 10% for validating our model. Lastly, we used Yolo and sent in our data to the neural network to use.
